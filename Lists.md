#### Source: [Lists](https://www.learnpython.org/en/Lists)

**Lists are very similar to arrays. they can contain any type of variable and they can contain as many variable as you wish**

```
mylist = []
mylist.append(1)
mylist.append(2)
mylist.append(3)

print(mylist)
print(mylist[2])

for x in mylist:
    print(x)
```
