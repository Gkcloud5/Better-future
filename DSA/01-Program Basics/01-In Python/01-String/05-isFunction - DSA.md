#### isdigit():

* It will check given string is only integer

```
str='123456'
str1="gokul"
print(str.isdigit())
print(str1.isdigit())
```

#### isalnum()

* Alphabet+number
* It will check given string have alphabet and number
	* If it have space then it will return false

```
str1="123"
str2="gokul"
str3="gokul 123"
print(str1.isalnum())
print(str1.isalnum())
print(str1.isalnum())
```
