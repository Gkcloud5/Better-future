#### Explanation about it:

* set is same as list but it did not allow duplicate value.
* We can simply create a set with `{}`

```
set1 = [1,2,3,4]
print(type(set1))

set2 = {1,2,3,4,1,3}
print(type(set2))
print(set2)
```
