* A file is generally storage of data.

**File operation takes place in the following order:
	* Open a file
	* Read or write
	* close the file**

##### Open a file:

 We use `open()` method to open files.

```
file1 = open("test.txt")
```

##### To give option:

```
file1 = open("test.txt", "r")
```
