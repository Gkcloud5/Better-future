##### nslookup:

* name server lookup tool is used to query name servers to find information about resource records.

```
nslookup google.com

Server:         127.0.0.53
Address:        127.0.0.53#53

Non-authoritative answer:
Name:   google.com
Address: 172.217.13.174
Name:   google.com
Address: 2607:f8b0:4020:805::200e

```
