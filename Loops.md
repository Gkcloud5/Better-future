#Basics #Loops 

Two types of loop:
	* For Loop
	* While Loop

#### For loop:
* For loop iterate over a given sequence
```
list = [1, 5, 7, 8]
for prime in list:
    print(prime)
```

##### range in for loop:
```
for x in range(5):
    print(x)
```


#### While loop:
* while loop repeat as long as a certain boolean condition is met.
