#### About System Call:

* System Call is a interface between process and OS
	* Process means programs during the execution

#### Types of System Calls:

* **Process Control:**
	* Create a process
	* Terminate a process
	* Load a process
	* Execute a process
	* Wait
	* signal -- acknowde
	* Allocate and free a memory
