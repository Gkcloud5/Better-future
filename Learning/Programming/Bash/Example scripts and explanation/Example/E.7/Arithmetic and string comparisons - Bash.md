##### Integer comparison:

* `-gt` --> is greater than
* `-ge` --> is greater than or equal to
* `-lt` --> is less than

##### String comparison:

* `=` --> is equal to
* `===` --> is qual to
* `!=` --> is not equal to
* `<` --> is less than 
	* ASCII alpha order
* `-z` --> true if string is null
