#### Explanation about it:

##### 1. /etc/profile:

* Bash reads `/etc/profile` instructions. this usually set the shell variables 
	* PATH
	* USER
	* MAIL
	* HOSTNAME


##### 2. /etc/bashrc:

* This file contains system wide definitions for shell functions and aliases
