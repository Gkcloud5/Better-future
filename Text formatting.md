#### Formatting Elements:

-   `<b>` - Bold text
-   `<strong>` - Important text
-   `<i>` - Italic text
-   `<em>` - Emphasized text
-   `<mark>` - Marked text
-   `<small>` - Smaller text
-   `<del>` - Deleted text
-   `<ins>` - Inserted text
-   `<sub>` - Subscript text
-   `<sup>` - Superscript text

#### Example:

```
<p><b>Gk-Hop</b><p>
<i>GK</i>
<p>My favorite color is <del>blue</del> red.</p>
```
