##### IP Address classes:

Class A: 1 - 126 (127 reserved for loopback and diagnostic test)
Class B: 128 - 191
Class C: 192 - 223
Class D: 224 - 239
Class E: 240 - 254


##### Private Addresses:

Class A: 10.0.0.0 through 10.255.255.255
Class B: 172.16.0.0 through 172.31.255.255
Class C: 192.168.0.0 through 192.168.255.255
